本外掛結合googlemap能用來做簡易的里程or時間->價格計算
適用於有需求將將里程或時間成本轉換成價格的網站
可用於網頁運輸類需求的開發或直接使用


== Description ==
推薦使用Elementor配合短代碼(shortcode)的方式使用本外掛
短代碼:[GMSC-plugin type=\'main_page\']

使用前請先閱讀教程:
[教學](https://codream.empathy.tw/wordpress/shipcost-with-googlemap-%e5%a4%96%e6%8e%9b%e8%a8%ad%e5%ae%9a%e6%95%99%e5%ad%b8/)

綠界模組的接收綠界回應網址請自行新增一個頁面，頁面屬性->範本
選擇 \"ECPay Template\"後直接發佈，設定的網址就填寫該頁面網址即可

本外掛推薦稍有程式概念的用戶使用
